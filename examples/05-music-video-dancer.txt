## KEY RULES (Smooth Camera + Sound)

1. Calculate clips: VIDEO DURATION ÷ CLIP LENGTH = Number of clips
2. Design ONE continuous camera move for the sequence, then segment
3. CRITICAL CONTINUITY: Clip N’s END FRAME = Clip N+1’s START FRAME (verbatim)
4. Smooth transitions: Blend all camera movement changes over 1–2 sec using easing curves
5. Use exact measurements: No approximations (~15% → 15%, ~3 ft → 3 ft)
6. Camera Motion Smoothness:

   * Transition Smoothing: Specify easing per movement change (linear, ease-in/out, sinusoidal)
   * Max Acceleration: ≤ 2 ft/sec² (linear)
   * Max Rotational Acceleration: ≤ 5°/sec²
   * Motion Sub-Segments: 1–2 sec intervals for gradual changes
   * Camera Inertia: 0–1 scale
   * Damping: Specify rotation / dolly damping (0–1)
   * Previous Clip Movement Vector: Track speed & rotation for continuity
7. Motion blur scale: 0–5% minimal, 5–15% subtle, 15–25% moderate, 25–40% strong, 40%+ extreme
8. Fill all fields per clip (camera, subject, background, lighting, focus, motion blur, style, tech, sound)

## CAMERA MOVEMENT QUICK REF (FLUID)

ROTATIONAL: Pan (horizontal), Tilt (vertical), Roll/Dutch (horizon tilt)
COMPLEX: Arc (circular around subject), Orbital (360°), Zoom (focal length)
SPEEDS:

* Glacial 0.1–0.3 ft/sec
* Slow 0.5–1 ft/sec
* Moderate 1.5–3 ft/sec
* Fast 4–6 ft/sec
* Rapid 7+ ft/sec

## PRO WORKFLOW WITH LAST FRAME METHOD

1. Generate Clip 1 as TEXT-TO-VIDEO
2. Export last frame (native resolution)
3. Generate Clip 2 as IMAGE-TO-VIDEO using Clip 1 last frame + Clip 2 prompt
4. Repeat for all clips
5. Stitch final sequence in editor
