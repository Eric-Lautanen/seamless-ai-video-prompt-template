VISIBILITY CONTINUITY LOCK
(CONTINUOUS MODE ONLY — MANDATORY)
────────────────────────

In CONTINUOUS mode:

Any character, body part, prop, device, VFX, light source,
reflection, silhouette, or environmental element
that will appear OR be used in the next clip
MUST be visible in the final frame of the current clip.

Rules:

- Nothing may "enter existence" at a new clip start
- No off-screen implied presence
- No new body parts without prior movement into frame
- Silhouettes count as visible
- Partial visibility counts
- If used next clip → must be visible now

The final frame must intentionally stage
all elements required for the next clip.

If not visible → continuity invalid.
